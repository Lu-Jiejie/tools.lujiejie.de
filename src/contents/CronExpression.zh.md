## 什么是 Cron 表达式

[**Cron 表达式**](https://zh.wikipedia.org/wiki/Cron)是一种用于配置定时任务的简洁语法。

它通过指定分钟、小时、日期、月份和星期等时间字段，定义任务应该**在什么时间**自动执行。

Cron 广泛应用于服务器运维、自动化脚本、数据备份、监控告警以及 CI/CD 等场景。

## 如何使用

- 选择常用预设，或通过可视化生成器配置时间规则。
- 设置分钟、小时、日期、月份和星期字段。
- 也可以直接输入 Cron 表达式。
- 查看自然语言说明，确认表达式含义是否正确。
- 查看接下来几次执行时间，验证调度结果。

## 字段顺序

标准五段 Cron 表达式按照以下顺序排列：

```text
* * * * *
│ │ │ │ │
│ │ │ │ └─ 星期（0–6）
│ │ │ └─── 月份（1–12）
│ │ └───── 日期（1–31）
│ └─────── 小时（0–23）
└───────── 分钟（0–59）
```

## 常用符号

| 符号 | 含义       | 示例        |
| ---- | ---------- | ----------- |
| `*`  | 任意值     | `* * * * *` |
| `,`  | 多个指定值 | `1,15,30`   |
| `-`  | 范围       | `1-5`       |
| `/`  | 间隔       | `*/10`      |

## 常见示例

| 表达式        | 含义               |
| ------------- | ------------------ |
| `*/5 * * * *` | 每 5 分钟执行一次  |
| `0 * * * *`   | 每小时执行一次     |
| `0 0 * * *`   | 每天 00:00 执行    |
| `0 9 * * 1-5` | 工作日 09:00 执行  |
| `0 0 1 * *`   | 每月 1 日执行      |
| `0 0 1 1 *`   | 每年 1 月 1 日执行 |
